# 🌸 Bloom To-Do List

A sleek, responsive task management app featuring modern "Bloom" typography and glassmorphism design.

## 🔗 Live Demo
[View Live Project Here](https://sabreenakhan.github.io/To-Do-List/)

## 📝 Description
Built during a 30-day JavaScript challenge, this application focuses on high-end UI/UX and efficient DOM manipulation. It allows users to add and remove tasks dynamically while maintaining a fully responsive layout for mobile and desktop devices.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox & Media Queries)
- **Logic:** Vanilla JavaScript (ES6+)
- **Icons:** Remix Icon Library

## ✨ Key Features
- **Dynamic CRUD:** Real-time adding and deleting of tasks.
- **Modern UI:** Custom gradient text-clipping and blur effects.
- **Mobile Optimized:** Fluid layouts using CSS breakpoints.
- **Clean Code:** Utilizes event delegation for performance.
